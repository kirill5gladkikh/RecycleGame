# Recycling Game

## Game Introduction

The Recycling Game is an educational game designed to teach players the importance of recycling and protecting the environment. Players sort different waste items into the correct recycling bins, such as paper, plastic, glass, and waste. The main goal of the game is to help players understand how proper waste sorting can reduce pollution and support environmental sustainability.

## Core Game Loop

The game starts from a main menu. After pressing the play button, the player enters the main gameplay scene, where waste items appear on the screen. The player must drag or place each item into the correct bin. Correct answers reward the player with points or coins, while wrong answers give negative feedback. After all items are sorted, the game shows a clear end screen, such as a level complete or game over screen. 
In the second level, the player also has to think about item contamination and material details. Some items, such as dirty plastic containers, must be cleaned with the water icon before they can be recycled correctly.

## How to Play

1. Start the game from the main menu.
2. Pick up or select a waste item.
3. Place the item into the correct recycling bin.
4. Earn points or coins for correct answers.
5. Lose points or receive feedback for wrong answers.
6. Learn the importance of correct recycling while playing.

## Educational Concept
The pedagogical goal of this game is to teach players that recycling is not only about matching items with bins by appearance, but also about understanding the condition and material of each item. Players learn how to sort waste into correct categories such as paper, plastic, glass, and general waste. The game also teaches that some items must be cleaned before they can be recycled. For example, a plastic food container with leftovers should not be placed into the plastic bin immediately. The player can use the water icon to clean it first, and only after that it becomes suitable for recycling.

The game mostly uses Behaviorism as its learning theory because players receive immediate feedback after every action. Correct choices reward the player with points or coins, while wrong choices give feedback and help the player understand their mistake. For example, if the player tries to put a greasy pizza box into the paper bin, the game explains that contaminated paper cannot be recycled. If the player tries to recycle a paper cup, the game gives feedback that many paper cups contain a plastic layer inside, so they are not always recyclable as normal paper.

These mechanics support learning through interaction, repetition, and feedback. Instead of simply reading recycling rules, the player actively tests different choices and learns from the results. This helps the player understand that proper recycling requires critical thinking, not just sorting items based on how they look.
