\# Recycling Game



\## About the Project



Game Introduction:



The Recycling Game is an educational game designed to teach players the importance of recycling and protecting the environment. Players learn how to correctly sort waste materials into recycling bins while improving their understanding of environmental sustainability.



The main goal of the game is to encourage people to recycle more often and become aware of how small actions can help reduce pollution and waste.

Also, teaching them the importance of sorting the trash in a correct order.



\---



\## Features



\- Interactive recycling challenges

\- Educational gameplay

\- Score tracking system

\- Different recyclable and non-recyclable items

\- Fun and simple user experience



\---



\## Purpose



This game was created to spread awareness about environmental issues and teach players how proper recycling can help keep the planet clean and healthy for future generations.



\---



\## How to Play



1\. Start the game

2\. Pick up or select waste items

3\. Place each item into the correct recycling bin

4\. Earn points for correct answers

5\. lose points for wrong answers

6\. Learn the importance of correct recycling while playing



\---



\## The applied learning theory: Mostly Behaviorism.



Behaviorism: The game constantly adjusts the points based on the correct placement of the trash into the bin, if the trash was thrown into the right bin +1 point otherwise -1.
